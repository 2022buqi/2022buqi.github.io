1.2

应用多元统计分析主要包括哪些分析方法？这些方法分别可以解决哪些方面的具体问题？

解答：

主要包括主成分分析、因子分析、聚类分析、判别分析、多元方差分析、典型相关分析、多元回归分析。

  1. 主成分分析： 用于降维，减少数据的维度，捕捉数据中的主要变异。 用于简化数据集，揭示主要特征。

  2. 因子分析： 识别和解释观察到的变量背后的潜在因子。

  3. 聚类分析 ： 将观察数据分组成具有相似特征的簇。

  4. 判别分析： 识别哪些变量在分离不同群体或类别中最具有判别性。

  5. 多元方差分析： 检验多个因变量在一个或多个自变量上的均值是否相等。用于比较多个群体的平均差异。

  6. 典型相关分析： 研究两组变量之间的最大相关性。

  7. 多元回归分析： 考虑多个自变量对一个或多个因变量的影响。在多变量情境下分析因变量和自变量之间的关系。

判别分析

4.1

简述欧几里得距离与马氏距离的区别与联系。

解答：

欧几里得距离是空间中两点之间的直线距离。对于两个n维空间中的点 x=(p1,p2,...,pn)Tx = (p_1, p_2, ..., p_n)^Tx=(p1​,p2​,...,pn​)T 和 y=(q1,q2,...,qn)Ty = (q_1, q_2, ..., q_n)^Ty=(q1​,q2​,...,qn​)T，平方欧式距离的计算公式为：

DE2(x,y)=(x−y)T(x−y)D_E^2(x,y) = (x - y)^T (x - y)DE2​(x,y)=(x−y)T(x−y)

马氏距离考虑了各个维度之间的相关性和协方差矩阵，平方马氏距离的计算公式为：

DM2(x,y)=(x−y)TΣ−1(x−y)D_M^2(x, y) = (x - y)^T \Sigma^{-1} (x - y)DM2​(x,y)=(x−y)TΣ−1(x−y)

其中，Σ\SigmaΣ 是协方差矩阵。

区别与联系：

  1. 计算方式： 欧几里得距离只考虑各个维度之间的距离，而马氏距离考虑了各个维度之间的相关性和协方差矩阵。

  2. 维度相关性： 欧几里得距离假设各个维度之间相互独立，而马氏距离能够捕捉维度之间的相关性。

  3. 应用场景： 欧几里得距离广泛应用于独立维度的场景，而马氏距离适用于考虑了维度相关性的情况，尤其在多维数据分析中有更多的应用。

在一定程度上，欧式距离是马氏距离的特殊情况，马氏距离是欧式距离的推广。

判别分析就是希望利用已经测得的变量数据，找出一种判别函数，使得这一函数具有某种最优性质，能把属于不同类别的样本点尽可能地区别开来。

4.3

简述距离判别法的基本思想和方法。

解答：

基本思想是通过测量样品到不同类别之间的距离，将样本判别为距离最近的类别。

方法：

  1. 样本距离计算： 对于每个未知样本，计算它与每个已知类别的样本之间的距离。

  2. 类别判别： 将未知样本判别为距离最近的已知类别。

  3. 距离度量： 选择合适的距离度量方法，如欧几里得距离、曼哈顿距离、切比雪夫距离等。

4.6

试分析距离判别法、贝叶斯判别法和费希尔判别法的异同。

解答：

不同之处：

  1. 基本思想：

    • 距离判别法： 使用距离度量来进行样本分类，通过样本点在特征空间中的位置和不同类别之间的距离进行分类。
    • 贝叶斯判别法： 基于贝叶斯定理，考虑先验概率和似然概率，计算后验概率来进行样本分类。
    • 费舍尔判别法： 通过投影，最大化类别间的方差，同时最小化类别内的方差，找到最佳投影方向。

  2. 计算方法：

    • 距离判别法： 主要涉及样本之间的距离计算，可使用不同的距离度量方法，如欧几里得距离、曼哈顿距离等。
    • 贝叶斯判别法： 利用贝叶斯公式，涉及先验概率和似然概率的估计，需要对分布进行建模。
    • 费舍尔判别法： 通过计算类别内散度矩阵和类别间散度矩阵，通过求解Fisher判别准则，得到最佳投影方向。
相同之处：

  1. 属于判别分析方法： 距离判别法、贝叶斯判别法和费舍尔判别法都属于判别分析的范畴，旨在找到一个合适的判别函数，将样本划分到不同的类别中。

  2. 用于多类别分类： 这三种方法都可以用于处理多类别分类问题，即将样本划分为两个或多个互斥的类别。

  3. 考虑了类别之间的差异： 在分类过程中，这三种方法都考虑了不同类别之间的差异，以便在特征空间中更好地区分不同的类别。

  4. 都属于监督学习： 这三种方法都是监督学习方法，它们的分类过程都需要已知类别的训练数据。

聚类分析

5.1

判别分析与聚类分析有何区别？

解答：

区别总结：

  • 监督 vs 无监督： 判别分析是一种监督学习方法，需要已知类别的标签；聚类分析是一种无监督学习方法，不需要事先标记的类别信息。

  • 目标： 判别分析关注如何最好地区分已知类别，以进行预测；聚类分析关注在不知道类别的情况下，如何发现样本之间的相似性，进行分组。

  • 应用领域： 判别分析常用于分类问题，例如预测；聚类分析常用于发现数据内在的结构，例如组织样本或发现子群。

5.2

试述系统聚类的基本思想。

解答：

距离相近的样品 (或变量)先聚成类,距离相远的后聚成类,过程一 直进行下去,每个样品 (或变量)总能聚到合适的类中。

5.5

试述K-均值法与系统聚类法的异同。

解答：

相同点：

  • K-均值法和系统聚类法一样，都是以距离的远近为标准进行聚类的。

不同点：

  • 系统聚类对不同的类数产生一系列的聚类结果，而K-均值法只能产生指定类数的聚类结果。

具体类数的确定，离不开实践经验的积累：有时也可以借助系统聚类法以一部分样品为对象进行聚类，其结果作为K-均值法确定类数的参考。

5.6

有序聚类法与系统聚类法有何区别？试述有序聚类法的基本思想。

解答：

K-均值法的基本思想是将每一个样品分配给最近中心（均值）的类中。系统聚类对不同的类数产生一系列的聚类结果，而K-均值法只能产生指定类数的聚类结果。具体类数的确定，有时也可以借助系统聚类法以一部分样品为对象进行聚类，其结果作为K-均值法确定类数的参考。

区别：

  1. 层次结构：

    • 系统聚类法通常生成一个层次结构，形成一棵聚类树。每一步可以看作是对数据的一次合并或划分，形成不同层次的簇。
    • 有序聚类法更强调聚类的顺序，将数据点一个个地添加到簇中，而不形成明确的层次结构。

  2. 类别的生成：

    • 系统聚类法的每一步都涉及到类别的合并或划分，形成整体的层次结构。
    • 有序聚类法则更注重每一步生成一个有序的类别，每个数据点依次加入。
有序聚类法的基本思想：

基本思想是按照一定的顺序将数据点逐个地添加到聚类中，形成有序的聚类序列。

主成分分析

6.1

试述主成分分析的基本思想。

解答：

通过对原始变量相关矩阵或协方差矩阵内部结构关系，利用原始变量的线性组合形成几个综合指标（主成分），在保留原始变量主要信息的前提下起到降维与简化问题的作用，使得在研究复杂问题时更容易抓住主要矛盾。

基本思想是通过线性变换将高维数据转换为低维数据，以保留数据中的主要信息。主成分分析旨在找到数据中的主成分，使得数据在这些主成分方向上的方差最大。
